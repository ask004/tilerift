# TileRift

Hybrid-casual mobile puzzle game concept (Tile Sort / Block Puzzle + Daily Challenge).

## Scope (MVP - 7 days)
- Core tile sort gameplay loop
- 60 levels from JSON
- Daily Challenge + streak
- Rewarded + Interstitial ads
- IAP: No Ads, Coin Pack
