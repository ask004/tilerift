# TileRift - Store Listing (EN)

## Title
TileRift: Fast Color Puzzle

## Short Description
Sort tiles in quick sessions and build your daily challenge streak.

## Long Description
TileRift is a short-session color sorting puzzle game built for mobile play. Move tiles to the right cells and complete each board within limited moves. Early levels teach the core loop quickly, while later levels reward careful planning.

Highlights:
- Smooth drag-and-drop and tap-to-move controls
- 60 levels with increasing difficulty
- Daily challenge and streak tracking
- Helpful boosters: Undo, Hint, Shuffle
- Offline-friendly gameplay loop

TileRift is designed for both quick breaks and consistent daily play.

puzzle, tile sort, color game, daily challenge, casual, logic
